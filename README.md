# Selenium-pytest-automation (Automação de Testes Web com Selenium + Pytest)

Projeto de automação de testes para interface web utilizando Selenium WebDriver e Pytest.

## O que é testado
- Login com credenciais inválidas (deve exibir mensagem de erro)
- Login com credenciais válidas (deve redirecionar para área segura)

## Tecnologias
- Python
- Selenium WebDriver
- Pytest

## Como executar
- pip install selenium pytest
- pytest test_login.py

**Para instalar antes de começar:**
- pip install selenium pytest
